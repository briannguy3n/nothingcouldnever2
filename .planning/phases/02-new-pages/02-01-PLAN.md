Create `_sass/_chooser.scss` with the following exact content. All styles scoped to `.chooser-page` to prevent global _base.scss bleed (per RESEARCH.md Pitfall 1). Values from approved UI-SPEC.

```scss
/*
  Chooser page styles — dark full-screen splash
  Scoped to .chooser-page to prevent global _base.scss color bleed
*/

.chooser-page {
  background: #000000;
  color: #ffffff;
  min-height: 100vh;
  margin: 0;

  // Override _base.scss global: a { color: $brand-color } where $brand-color = black
  // Without this, links are black-on-black — invisible on dark background
  a {
    color: #ffffff;
    text-decoration: none;
  }
}

.chooser-container {
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
  min-height: 100vh;
  gap: 16px;
}

.chooser-primary {
  font-size: 4.5rem;
  font-weight: 700;
  color: #ffffff;
  text-decoration: none;
  transition: color 0.2s ease;

  &:hover {
    color: #aaaaaa;
  }
}

.chooser-secondary {
  font-size: 1rem;
  font-weight: 400;
  color: #888888;
  text-decoration: none;
  transition: color 0.2s ease;

  &:hover {
    color: #cccccc;
  }
}

// Mobile breakpoint per UI-SPEC
@media (max-width: 480px) {
  .chooser-primary {
    font-size: 3rem;
  }

  .chooser-container {
    gap: 8px;
  }
}
```

Per D-01: ultra-minimal, no branding elements.
Per D-04: pure black #000 background, white text.
Per D-08 area hover values from UI-SPEC Component Contracts: "n0thing" hover fades to #aaaaaa, "other stuff" hover fades to #cccccc.
